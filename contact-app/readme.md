
# 📇 Contact Management App (Python CLI)

A simple and efficient Contact Management Application built using Python. This CLI-based tool helps you manage your contacts with features like create, view, update, delete, search, count, and exit.

## 🚀 Features

1. **Create a Contact** – Add a new contact with details like name, phone number, email, etc.
2. **View Contacts** – Display all saved contacts in a clean format.
3. **Update Contacts** – Modify existing contact information.
4. **Delete Contacts** – Remove a contact from the list.
5. **Search Contacts** – Find contacts by name, phone number, or other fields.
6. **Count Contacts** – Show the total number of saved contacts.
7. **Exit** – Quit the application safely.

## 🛠️ Technologies Used

- **Python 3**
- File handling (for storing contacts locally)
- Optional: `pickle`, `json`, or `csv` for persistent data storage

## ✅ How to Run

1. Clone this repository or download the files.
2. Make sure Python 3 is installed.
3. Run the application:

```bash
python app.py
```
